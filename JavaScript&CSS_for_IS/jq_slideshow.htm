Itdev160 Slide Show Assignment

We will be learning the wonderfull advantages of using jQuery to create a slideshow. There are two good videos in the jQuery unit on our class CD. By following the 2 videos you should be able to create a simple slideshow.

Procedure

  1. View the JS Slideshow and the jQuery Slideshow videos in the jQuery unit of our class cd
  2. The videos will show you how to create the code to create a slideshow
  3. Pick out 4 images from the Halloween folder in the DataFilesforStudents folder
  4. Put a link to it on the homepage of the Final Project Home page
  5. Follow these quidelines in doing this project
    • Use a width of 260 and a height of 300 for the slide
    • Make up a short description of each image
    • Use jQuery to actually run the show
