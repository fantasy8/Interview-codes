Experimenting with the advice from these sites I came up with this layout sample that we can use to recreate the Windows 7 project.

Column One Column One Column One Column One Column One Column One Column One Column One Column One Column One Column One Column One
Column Two Column Two Column Two Column Two Column Two Column Two Column Two Column Two
Note: you will have to span tags to change the size of the the tip number to 24pt and to change the size of the tip title to 14pt.

Here is the html markup 
#container { 
    width: 650px; 
    border: solid;  border-width: thin; padding-right: 10px;
    padding-left: 10px; padding-top: 5px; padding-bottom: 10px;
}
#col_one { 
    width: 290px;
    margin: 10px;
    float: left;
    background-color: yellow;
    text-align: justify; 
} 
#col_two { 
    width: 290px;
    margin: 10px;
    float: left;
    background-color: azure;
    text-align: justify;   
} 

<div id="container"> 

   <div id="col_one">Column One Column One Column One 
   Column One Column One Column One Column One Column One 
   Column One Column One Column One Column One</div> 

   <div id="col_two">Column Two Column Two Column Two 
   Column Two Column Two Column Two Column Two Column Two</div> 

</div>
